# Contributing to PromptPatterns

Thank you for considering contributing!

## How to contribute

1. Fork the repository
2. Create a new branch:git checkout -b feature-name
3. Add your improvement (new pattern, fix, or example)
4. Commit your changes:git commit -m "Add: description of change"
5. Push and open a Pull Request

## Guidelines

- Keep patterns clear and practical
- Include examples
- Avoid overly complex explanations
- Follow existing formatting style
